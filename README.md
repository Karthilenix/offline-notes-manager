# Offline Notes Manager

A browser-based offline-first notes application built using Vue 3 and IndexedDB.

## Features

- Create Notes
- Edit Notes
- Delete Notes
- Search Notes
- Sort Notes
- Export All Notes as JSON
- Export Single Note
- Dark Mode
- Responsive Layout
- Offline Storage using IndexedDB

## Tech Stack

- Vue 3
- Vite
- IndexedDB
- idb

## Installation

```bash
cd frontend
npm install
npm run dev
